컨텐트 테이블

서문

이 매뉴얼은 톰캣 프로젝트 개발자 커뮤니티의 많은 멤버들로부터 기여를 받았습니다. 다음의 저자는 상당한 컨텐트를 제공하였습니다:

  • 크레이그 R. 맥클라나한 (craigmcc@apache.org)
컨텐트 테이블

제공되는 정보는 다음과 같은 부문으로 나뉘어져 있습니다.:

  • 소개 - 여기에서는 링크와 다른 소스 정보를 참조를 포함한 정보를 간단하게 설명하고 있습니다.
  • Installation - 웹 어플리케이션 개발을 위한 톰캣을 사용하기 위해 요구되는 소프트웨어 컴포넌트의 습득과 설치에 대해 다루고 있습니다.
  • 개발 구조 - 웹 어플리케이션을 위한 표준 디렉토리 구조, (서블릿 API 사양에 정의된), 웹 어플리케이션 배포 서술자 , 그리고 톰캣을 당신의 개발환경에 통합시키기 위한 옵션에 대해 설명하고 있습니다.
  • 소스 구조 - 당신 프로젝트의 디렉토리를 이용하여 소스코드를 효율적으로 구성하는 방법에 대해서 설명합니다. 그리고 앤트에서 build.xml을 사용하여 컴파일을 관리하는 것에 대해 소개합니다.
  • 개발 과정 - 권장되는 배포방법과 소스 구조를 활용한 전형적인 개발 과정에 대한 간략한 설명을 제공합니다.
  • 예제 어플리케이션 - 이번 항목은 이 메뉴얼에 설명된 원칙을 따르는, 매우 간단하지만 기능적으로는 완성된 "Hello, world" 어플리케이션을 포함합니다. 이 애플리케이션을 설명된 기법을 이용하여 연습하는데 사용할 수 있습니다.
