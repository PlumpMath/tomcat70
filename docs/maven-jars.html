Using Tomcat libraries With Maven
Tomcat Snapshots
톰캣 스냅샷은 아파치 스냅샷 저장소에 위치합니다. 공식적인 URL은 
http://people.apache.org/repo/m2-snapshot-repository/org/apache/tomcat/
입니다. 스냅샷은 정기적은 아니지만 변경사항이 있고, 톰캣 팀이 새로운 스냅샷이 유용하다고 판단하면 주기적으로 만들어집니다.
Tomcat Releases
안정 릴리스는 센트럴 메이븐 저장소로 퍼블리쉬됩니다. 이에 대한 URL은 
http://repo2.maven.org/maven2/org/apache/tomcat/
입니다.
