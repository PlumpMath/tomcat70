mod_proxy와 아파치 HTTP 서버 2.x 사용하기
아파치 HTTP 서버 2.2에 대해 mod_proxy 설명서를 참조하십시오. 이것은 HTTP 또는 AJP로드 밸런싱 중 하나를 지원합니다. mod_proxy의 새로운 버전은 아파치 HTTP 서버 2.0을 사용할 수 있지만 mod_proxy에서 아파치 HTTP 서버 2.2는 코드를 사용하여 별도로 컴파일해야 합니다.
