WebSocket How-To

Table of Contents
  • Overview
  • Application development
Overview

톰캣은 RFC 6455에 정의된 WebSocket을 제공합니다. 이 기능은 아직 완료되지 않았으며 버그 리포트 양식에 따른 (Bugzilla를 통한) 여러분의 피드백과 API 변경(dev list를 통한) 제안, 또는 의견(dev list를 통한)을 받고 있습니다.

Application development

어플리케이션 개발을 위한 API는 아직 완료되지 않았습니다. 일정보다 빠르게 보려면 여기의 문서보다 org.apache.catalina.websocket 패키지의 Javadoc를 보길 권장합니다. Javadoc 패키지들은 Tomcat 바이너리 배포판을 포함하지 않습니다. 관련하여 보길 원한다면 여러분은 'Full documentation' 배포판을 다운로드 후 인스톨하거나 소스로부터 빌드해야 합니다. 여러분은 Apache Tomcat 사이트를 통해서 읽을 수도 있습니다. WebSocketServlet 클래스에서 시작하세요.

또한, WebSocket API를 사용하는 여러 예제 어플리케이션들이 있습니다. 예제들은 클라이언트인 html과 서버쪽 코드 code 양쪽을 보셔야합니다.

API는 유동적이며 포인트 릴리즈 사이에 크게 변경될 수 있다는 것을 명심하세요. 문서는 안정화를 고려하여 한번 업데이트 될것입니다.(JSR 356이 완료되기 전에 될 가능성이 있음).
