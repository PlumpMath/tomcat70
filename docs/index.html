문서 인덱스

Introduction

이 페이지는 아파치 톰캣 서블릿/JSP 컨테이너에 대한 문서 번들의 최상위 엔트리 지점입니다. 아파치 톰캣 버전 7.0은 자바 커뮤니티 프로세스에서 나온 서블릿 3.0 및 JSP(JavaServer Pages) 2.2 사양을 구현했고, 웹 응용 프로그램 및 웹 서비스의 개발 및 배포를 위한 유용한 플랫폼으로 만드는 많은 추가 기능을 제공합니다.

탐색 메뉴에서 링크 중 하나를 (왼쪽)를 선택하면 더 자세한 설명서가 나옵니다. 사용 가능한 각 사용 설명서는 아래에 자세히 설명되어 있습니다.

Apache Tomcat User Guide

다음 문서는 아파치 톰캣 7을 다운로드하고 설치하고, 많은 아파치 톰캣 기능을 사용하는 데 도움이 될 것입니다

  1. 소개 - 간략하고, 전반적인 아파치 톰캣 개요.
  2. 설치 - 다양한 플랫폼에 아파치 톰캣을 설치하고 실행하는 방법.
  3. 첫번째 웹 애플리케이션 - 서블릿 스펙에 정의된 웹 애플리케이션의 컨셉에 대한 소개. 웹 애플리케이션 소스 트리의 기본 구조와 웹 애플리케이션 아카이브의 구조, 웹 애플리케이션 배포 설명자 소개를 커버합니다.(/WEB-INF/web.xml).
  4. 디플로이어(Deployer) - 웹 애플리케이션의 배포, 프리컴파일, 검증하기 위한 아파치 톰캣 디플로이어 동작법.
  5. 매니저 - 아파치 톰캣이 실행되는 동안 웹앱 배포, 삭제, 재배포하기 위한 매니저의 동작법.
  6. 렐름(Realm)과 접근 제어 - 컨테이너 관리 보안을 이용하는 웹 애플리케이션에서 사용할 렐름(사용자, 비밀번호, 관련된 역할의 데이터베이스)을 설정하는 방법을 설명.
  7. 보안 관리자 - 웹 애플리케이션의 동작을 세세하게 제어하는 자바 보안 관리자를 설정하고 사용하는 방법.
  8. JNDI 자원(Resources) - 각 웹 애플리케이션별로 제공되는 JNDI 네이밍 컨텍스트에서 표준과 사용자 자원을 설정하는 법.
  9. JDBC 데이터소스 - DB 커넥션 풀과 JNDI 데이터소스 설정하기. 많은 인기있는 데이터베이스에 대한 예제.
  10. 클래스로딩 - 아파치 톰캣에서 클래스 로딩에 대한 정보, 애플리케이션 클래스들이 어디에 위치해야 인식하는지를 포함.
  11. JSP - JSP 컴파일러 사용법과 자스퍼 설정에 대한 정보.
  12. SSL - https 프로토콜을 사용하는 요청을 처리하기 위해 SSL 지원을 설치하고 설정하기.
  13. SSI - 아파치 톰캣에서 서버측 인클루드 사용하기.
  14. CGI - 아파치 톰캣에서 CGI 사용하기.
  15. 프록시 지원 - 프록시 서버 (또는 프록시 서버 기능을 하는 웹 서버) 뒤에서 실행하는 아파치 톰캣 설정하기.
  16. MBean 설명자 - 사용자 정의 구성 요소에 대한 MBean 설명자 파일을 설정하기.
  17. 기본 서블릿 - 기본 서블릿 및 디렉토리 목록 사용자 정의 설정하기.
  18. 아파치 톰캣 클러스터링 - 아파치 톰캣 환경에서 세션 복제를 활성화.
  19. 밸런서 - 로드 밸런서 애플리케이션을 설정, 사용, 확장하기.
  20. 커넥터 - 아파치 톰캣에서 가능한 커넥터와 네이티브 웹 서버 통합.
  21. 모니터링과 관리 - JMX 원격 지원을 활성화하고, 아파치 톰캣을 모니터링하고 관리하는 도구 사용하기.
  22. 로깅 - 아파치 톰캣에서 로깅 구성.
  23. 아파치 포터블 런타임 - 뛰어난 ​​성능, 확장성 및 네이티브 서버 기술로 더 나은 통합을 제공하기 위해 APR 사용하기.
  24. 가상 호스팅 - 아파치 톰캣에서 가상 호스팅 설정하기.
  25. 고급 IO - 정규, 블로킹 IO 위에서 가능한 확장.
  26. 추가 구성 요소 - 추가, 옵션 구성 요소를 구하기.
  27. 메이븐으로 톰캣 라이브러리를 사용 - 메이븐을 통해서 톰캣 jar 구하기.
  28. 보안 고려 사항 - 아파치 톰캣 설치를 안전하게 할 때 고려해야 할 옵션..
  29. 윈도우 서비스 - 마이크로소프트 윈도우의 서비스로 톰캣을 실행.
  30. 윈도우 인증 - 통합된 윈도우 인증을 사용하도록 구성하기.
  31. 높은 동시성 JDBC 풀 - 대안 JDBC 풀을 사용하도록 톰캣을 구성하기.
  32. 웹소켓 지원 - 아파치 톰캣을 위한 웹소켓 애플리케이션을 개발하기.
